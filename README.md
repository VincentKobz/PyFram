# PyFram

PyFram is a small implementation of WallFramAlpha with Python, Heroku and Django.

## Mathematical calculations:

basic, function, normalize, cos, sin, arccos, arcsin

Examples:
* `basic: 1+1`
* `function: x + 2 : {x value}`
* `normalize: 2, 5`
* `cos: {degree}`
* `sin: {degree}`
* `arccos: {degree}`
* `arsin: {degree}`
